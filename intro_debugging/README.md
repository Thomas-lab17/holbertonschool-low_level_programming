# Debugging Project

A low-level programming project focused on debugging techniques in C.

## Description

This project covers fundamental debugging concepts and practices for C programs, including identifying and fixing common errors.

## Author

Thomas

## Learning Objectives

- Understand common debugging tools
- Identify and fix bugs in C code
- Practice error handling and validation

## Requirements

- C compiler (gcc)
- Basic C knowledge

## Usage

Compile and run the C programs to explore debugging techniques.

```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o program
./program
```
