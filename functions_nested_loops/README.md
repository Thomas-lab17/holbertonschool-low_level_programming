# Functions and Nested Loops

This project covers fundamental C programming concepts including function definitions, function calls, nested loops, and control flow structures.

## Learning Objectives

- Understand function declaration and implementation
- Master nested loop patterns and iterations
- Learn about variable scope and parameters
- Practice debugging and code organization

## Project Structure

This directory contains exercises focused on:
- Writing and calling functions
- Using loops within loops
- Working with loop conditions and break statements
- Understanding function return values

## Getting Started

Compile C files using:
```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o program
```
